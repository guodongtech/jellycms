自定义标签 - 模板标签

适用范围
全站任意地方均可使用
标签作用
用于调取自定义标签信息
调用方式 
					// 更多value值调用请看下方可调用字段
		    		{label:标签名 value=description} 例：{label:testlabel value=description}即可获取testlabel标签的标签描述
可调用字段
序号 字段名 调用 注释
id
{label:标签名 value=id}
自定义标签ID
name
{label:标签名 value=name}
标签名称
value
{label:标签名 value=value}
标签值
type
{label:标签名 value=type}
标签类型
description
{label:标签名 value=description}
标签描述
content
{label:标签名 value=content}
标签内容
自定义标签名称可以有字母数字下划线组合 必须以字母开始且不以下划线结尾
type为标签类型包括input(单行文本)、textarea(多行文本)、radio(单选)、checkbox(复选)、pic(图片上传)、file(附件上传)、time(日期选择)、editor(编辑器)、select(下拉选择)等
value字段为标签值，只有radio、checkbox、select需要设置标签值，比如创建一个sex标签，type为radio,选项为男和女，那么{label:sex value=value}输出‘男,女’。
content字段为标签内容，当type为radio、checkbox、select时，content为选中的内容，多个选中的值以半角逗号连接
添加自定义字段
我们添加了常用的内容字段，但并不一定能满足你的开发需求。你可以非常轻松地添加自定义字段。

1.在数据库label表添加自定义字段

推荐使用phpmyadmin或navicat for mysql对数据库进行操作

2.在模板里添加自定义字段表单项

模板位置：\app\admin\Views\admin\html\label.html
此处我们使用是js动态回填，你不必为自定义字段赋值
后台表单也无需处理，系统会自动将新添加字段值保存到数据库

3.调用自定义字段

{label:标签名 value=自定义字段}
