广告位标签 - 模板标签

适用范围
全站任意地方均可使用
标签作用
用于调取广告位信息
调用方式 
					// 更多value值调用请看下方可调用字段
		    		{myad:标识 value=name} 例：{myad:testmyad value=name}即可获取testmyad标签的标签名称
可调用字段
序号 字段名 调用 注释
id
{myad:标识 value=id}
广告位ID
name
{myad:标识 value=name}
广告位名称
label
{myad:标识 value=label}
广告位标识
type
{myad:标识 value=type}
广告位类型（文本、单图、多图轮播）
timeset
{myad:标识 value=timeset}
期限（等于0为永不过期）
start_time
{myad:标识 value=start_time}
开始时间（timeset等于1时有效）
end_time
{myad:标识 value=end_time}
结束时间（timeset等于1时有效）
normbody
{myad:标识 value=normbody}
广告内容（包含js及css样式）
expbody
{myad:标识 value=expbody}
过期显示内容
path
{myad:标识 value=path}
图片地址（多图以半角逗号连接）
link
{myad:标识 value=link}
文本及图片跳转地址（多个地址以半角逗号连接，与path对应）
desc
{myad:标识 value=desc}
图片描述（多个描述以半角逗号连接，与path对应）
width
{myad:标识 value=width}
图片及轮播图宽度
height
{myad:标识 value=height}
图片及轮播图高度
text_size
{myad:标识 value=text_size}
文字大小（仅文本类型有效）
text_color
{myad:标识 value=text_color}
文字颜色（仅文本类型有效）
text_content
{myad:标识 value=text_content}
文本内容（仅文本类型有效）
广告类型分为text（文本）、single_img（单图）和more_img（多图轮播）三中类型
当类型为多图轮播或单图时，path、link、desc、width、height 才会生效
当类型为文本时，text_size、text_color、text_content 才会生效
添加自定义字段
我们添加了常用的内容字段，但并不一定能满足你的开发需求。你可以非常轻松地添加自定义字段。

1.在数据库myad表添加自定义字段或者在模型里添加扩展字段

推荐使用phpmyadmin或navicat for mysql对数据库进行操作

2.在模板里添加自定义字段表单项

模板位置：\app\admin\Views\admin\html\myad.html
此处我们使用是js动态回填，你不必为自定义字段赋值
后台表单也无需处理，系统会自动将新添加字段值保存到数据库

3.调用自定义字段

{myad:标识 value=自定义字段}
